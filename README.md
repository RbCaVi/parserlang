# Parserlang

Name might change.

A programming language with backtracking.

Compiled to a VM.

All data types are immutable.

Functions are first-class values.

Data types:
* null
* boolean
* integer
* float
* string
* array
* object
* closure
* generator
* any user defined types

pv is copied from [jq](https://github.com/jqlang/jq)
